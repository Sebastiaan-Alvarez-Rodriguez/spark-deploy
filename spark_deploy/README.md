# spark-deploy
Framework and CLI-tool to install Spark clusters on remote clusters. 

## Requirements
 - Python>=3.2
 - remoto>=1.2.0
 - metareserve>=0.1.0


## Installing
Simply execute `pip3 install . --user` in the root directory.


## Usage
To `start` a cluster, use:
```bash
spark-deploy start
```

For more information, use:
```bash
spark-deploy -h
```
